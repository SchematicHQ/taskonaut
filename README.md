# Schematic-ECS-EXEC - see

## Pre-requisites

- Node.js
- npm
- aws-cli
- session-manager-plugin

## install

```bash
npm install -g .
```

## usage

```bash
aws sso login --profile <profile>
export AWS_PROFILE=schematic-prod
export AWS_REGION=us-east-1
see --help
see
```

## Remove

```bash
npm remove -g see
```
